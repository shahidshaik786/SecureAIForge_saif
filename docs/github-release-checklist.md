# GitHub Release Checklist

- [ ] Remove secrets from `.env`
- [ ] Confirm `.gitignore` excludes evidence/reports/logs
- [ ] Run tests
- [ ] Run lint if available
- [ ] Run doctor
- [ ] Start dashboard locally
- [ ] Capture screenshots
- [ ] Update README screenshots
- [ ] Confirm disclaimer
- [ ] Confirm license
- [ ] Commit
- [ ] Push to GitHub
